Que es React

Es una biblioteca (libreria) de Javascript de codigo abierto desenada para crear interfaces de usuario. Es muy usado actualmente. react nos da herramientas para crear nuestra aplicacion web.

Ventajas de React

  • facil de aprender y usar
  • Crear componentes reutilizables
  • crear aplicaciones dinamicas, cambian el estado de la aplicacion
  • buen desempeno, cada cambio que se hace, se actualiza a nivel componente.

Conceptos Basicos de React

  • Componente: Parte de la interfaz de usuario que es independiente(estado y funcionalidad especifica) y reusable(Lo escribimos una sola vez y lo usamos varias veces)
    • Componenetes funcionales: Es una funcion de Javascript, esta funcion retorna un elemento de React (JSX)

      • Siendo props = "propiedades", son argumentos que pueden recibir los compoenentes react, los props solo pueden ser enviados de padre a hijo

        Caracteristicas

      • Debe retornar un elemento de react JSX
      • El nombre debe comenzar con una letra mayuscula
      • Puede recibir valoreso props si es necesario
    • Componenetes de clase: Es una clase de ES6 que retorna un elemento JSX, su sintanxis es mas elaborada.

        • Caracteristicas

      • Debe extender de React.Component
      • Debe tener un metodo render para retornar un elemento JSX
      • Puede recibir valores o props si es necesario
  • Estado:(State) Representacion en Javascript del conjunto de propiedades de un componenee y sus valores actuales
  • Hooks o anzuelos: Nos permiten agregar cierto estado a los compoenentes funcionales.Ahora si podemos actualizar los estados de un componente funcional.
    "Funcion Especial que te permite trabajar con estados en componentes funcionales y otros aspectos de react, sin escribir componentes de clase"
  • Event Listener: Funcion en javascript que es ejecutada cuando ocurre un evento especifico

Introduccion a JSX

Es una extension de react para la sintaxis de javascript, nos permite describir como se veran los componentes similar a HTML, combinando asi js y HTML. Recordar que JSX es opcional. En React podemos usar expresiones javascript normales con JSX.

    Ventajas

  • Estructura mas facil de visualizar
  • Errores y advertencias mas utiles

Elementos JSX: unidades mas basica que se puede crear, los compoenentes estan conformados de elementos, y no al contrario.

Atributos: Se pueden agregar como en html, id, clases entre otro.

Estructura

Iniciando con React JS

  1. Instalar node js en la computadora, basicamente nos permite ejecutar codigo javascript fuera de un brouser
  2. Crear el proyecto react con el comando "npx create-react-app nombre-app"
  3. Enconces seguimos los pasos que recomienda y ponemos npm start en consola para iniciar el servidor
